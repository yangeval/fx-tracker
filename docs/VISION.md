# Vision: FX-Tracker (가칭)

## 1. Project Name
FX-Tracker (환율 투자 현황 관리 시스템)

## 2. Core Value
분산된 플랫폼(토스, 스위치원, 카카오뱅크)의 환율 투자 현황을 모바일과 PC에서 지연 없이 즉각적으로 확인하고 관리한다.

## 3. Problem Statement
- **노션 관리의 한계**: 노션 DB의 모바일 뷰는 로딩이 매우 느리고, 화면 규격에 최적화되지 않아 원하는 정보를 찾기 위해 과도한 스크롤과 조작이 필요함.
- **가시성 부족**: 엑셀 방식의 리스트 나열은 현재 진행 중인 핵심 계약 정보(매수/매도가)를 한눈에 파악하기 비효율적임.
- **수동 관리의 피로**: API 미지원으로 인한 수동 입력 과정에서, 입력을 위한 동선이 길고 UI가 무거워 입력 자체가 스트레스가 됨.

## 4. Main Features
- **멀티 플랫폼 통합 대시보드**: 토스, 스위치원, 카카오뱅크 등 분산된 은행 앱의 계약 상태를 실시간 동기화 DB를 통해 통합 표시.
- **정밀 자산 현황 요약**:
    - **보유 자산**: 이미 매수 완료된 총 원화 환산 금액.
    - **운용 자산**: 매수 주문을 걸어놓았으나 아직 체결되지 않은 원화 환산 금액.
    - **기대 수익**: 보유 중인 계약이 목표가에 매도될 경우의 예상 총수익 합산 표출.
- **진행 계약 최적화 뷰**: 모바일에서도 [플랫폼 | 매수가 | 목표가 | 수량] 정보를 한눈에 파악할 수 있는 카드/테이블 뷰 제공.
- **통합 히스토리 매니저**: 종료 여부와 상관없이 모든 거래 내역을 보관하며, 다차원 필터링 및 정렬 기능을 통해 과거 데이터 분석 지원.
- **계약 상태 워크플로우**: [체결대기 -> 보유 중 -> 매도완료] 상태를 원클릭으로 변경하여 관리 편의성 극대화.
- **크로스 플랫폼 동기화**: Supabase 등 웹 기반 DB를 활용하여 PC와 모바일 간의 완벽한 데이터 일관성 유지.

## 6. Expected Outcome
- **빠른 판단력**: 화면 로딩 대기 시간 없이 즉시 계약 현황을 확인하여 환율 변동에 빠르게 대응 가능.
- **사용자 경험 개선**: 노션의 답답한 로딩에서 벗어나 쾌적하고 반응 속도가 빠른 전용 앱/웹 환경 제공.
- **관리 효율화**: 데이터 입력 및 조회에 들어가는 물리적 시간을 단축하여 투자의 본질에 집중할 수 있음.
